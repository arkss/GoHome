### .gitignore

https://gitignore.io/ 에 접속하셔서 본인의 OS, IDE, framework 에 맞는 설정을 검색 후 `.gitignore` 에 추가하고 push해주세요.

### docs

각 분야 dir에는 `docs` 라는 dir가 존재합니다.

해당 dir에는 해당 분야에 대해 필요한 문서 등을 저장해주시면 됩니다.

ex) db쪽 같은 경우에는 백업 sql, 더미데이터 insert sql 등
